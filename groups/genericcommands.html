Most standard Lua functions can be used, excluding functions that execute code (use include instead) or access files (use file. instead)
print(message:string)
Displays the message in the console, also used to respond to rcon commands
Returns [nil]
include(file:string)
Paths are relative to the current script's location. To use root paths (from GameData), start the path with '/'
Returns [bool] of success
PrintTable(tbl:table)
Displays the contents of the specified table in the console
Returns [nil]
