file.Read(file:string)
Returns contents of a file (uses glua/data as starting path)
Returns [string]

file.Write(file:string, buffer:string)
Writes to a file (uses glua/data as starting path)
Returns [nil]

file.Exists(file:string)
Checks if a file exists (uses glua/data as starting path)
Returns [bool]

file.ListFiles(path:string, extension:string)
Returns a string table listing all of the files in the path. extension arg is optional. (uses glua/data as starting path)
Returns [string] table
