json.Register(file:string)
Returns a jsonHandle (or in other words, the pointer to the root JSON node) for the given file, if it is a valid JSON file.
Returns [jsonHandle] or [nil] on failure
json.GetObjectItem(root:jsonHandle, nodeName:string)

Returns [jsonHandle]
json.Clear(nil)
Run this func once you're done with json stuff.
Returns [nil]
json.GetFirstObject(node:jsonHandle)

json.GetNextObject(node:jsonHandle)
Returns the next object that you're targetting. Returns nil if no node is found.
Returns [jsonHandle] or [nil] on failure
json.IsNumber(node:jsonHandle)

json.IsObject(node:jsonHandle)
Returns true if the node is an object type
Returns [bool]
json.IsBoolean(node:jsonHandle)
